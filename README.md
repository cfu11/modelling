Please visit the attached file above for Python script. 
This project models hydrogel diffusion using Python. Hydrogels are polymer networks commonly used in drug delivery, tissue engineering, and biomaterials applications. Understanding how molecules diffuse through these networks is important for predicting transport behavior and release kinetics.

Libraries used:
- NumPy
- Matplotlib.pyplot

Outcome:
The script generates a graph showing the evolution of solute concentration across the hydrogel domain.
